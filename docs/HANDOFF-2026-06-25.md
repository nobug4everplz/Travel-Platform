### 6.2 DB 改動（init-db.php）

```sql
-- users 加欄位
ALTER TABLE users ADD COLUMN tier VARCHAR(20) DEFAULT 'free';
ALTER TABLE users ADD COLUMN tier_expires_at TIMESTAMP NULL;
ALTER TABLE users ADD COLUMN ai_tokens_daily INT DEFAULT 10000;
ALTER TABLE users ADD COLUMN ai_tokens_used INT DEFAULT 0;
ALTER TABLE users ADD COLUMN tokens_date DATE NULL;

-- 方案定義表
CREATE TABLE subscription_plans (
    id SERIAL PRIMARY KEY,
    slug VARCHAR(30) UNIQUE,
    name VARCHAR(50),
    price_monthly INT,
    price_yearly INT,
    ai_tokens_daily INT,
    trip_draft_limit INT,
    trip_publish_limit INT,
    has_analytics BOOLEAN DEFAULT false,
    has_badge BOOLEAN DEFAULT false,
    has_featured BOOLEAN DEFAULT false,
    is_active BOOLEAN DEFAULT true
);

-- 訂單記錄表
CREATE TABLE subscription_orders (
    id SERIAL PRIMARY KEY,
    user_id INT REFERENCES users(id),
    plan_id INT REFERENCES subscription_plans(id),
    amount INT,
    status VARCHAR(20) DEFAULT 'pending',
    payment_method VARCHAR(30),
    paid_at TIMESTAMP,
    starts_at TIMESTAMP,
    expires_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 8. 已知注意事項

| 項目 | 說明 |
|------|------|
| Hermes secret masking | 寫入含 API key 的檔案會被截斷，需用 `execute_code`+subprocess 繞過 |
| Flycast DNS | 不穩定，已改用 `.internal` DNS |
| fail counter lockout | 連續 5 次 AI 錯誤鎖 10 分鐘，session 級 |
| DeepSeek tools schema | `properties` 必須是 `{}`（object），不能是 `[]`（array） |
| PG 語法 | 專案原是 MySQL，遷移後殘留需注意：`INSERT IGNORE`、`DISTINCT+ORDER BY`、boolean vs SMALLINT |
